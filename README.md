# Gold Zakat Calculator

Gold Zakat Calculator is an Android application developed using Java in Android Studio.

## Features

- Calculate gold zakat based on weight and current gold value
- Keep Gold (Uruf = 85g)
- Wear Gold (Uruf = 200g)
- Input validation and error handling
- Share application repository
- About page with developer information
